# 구글 폼 연동 가이드

## 대기자 리스트 구글 폼 만들기

### 1단계: 구글 폼 생성

1. **구글 드라이브 접속**
   - https://drive.google.com 접속
   - 구글 계정으로 로그인

2. **새 폼 만들기**
   - 좌측 상단 "새로 만들기" 클릭
   - "Google Forms" 선택
   - 또는 직접 https://forms.google.com 접속

3. **폼 제목 설정**
   - 폼 제목: "한국어 프로그램 대기자 리스트" 또는 "Korean Program Waitlist"
   - 설명 추가 (선택사항)

### 2단계: 필드 추가

다음 3개의 필드를 순서대로 추가하세요:

#### 필드 1: 이름 (Name)
- 질문 유형: "단답형" (Short answer)
- 질문: "이름을 입력해주세요" / "Please enter your name"
- 필수 항목: ✅ 체크

#### 필드 2: 이메일 (Email)
- 질문 유형: "단답형" (Short answer)
- 질문: "이메일 주소를 입력해주세요" / "Please enter your email address"
- 필수 항목: ✅ 체크
- 응답 검증: ✅ 체크
- 검증 유형: "이메일" 선택

#### 필드 3: 언어 레벨 (Language Level)
- 질문 유형: "드롭다운" (Dropdown) 또는 "객관식" (Multiple choice)
- 질문: "현재 한국어 레벨을 선택해주세요" / "Please select your current Korean level"
- 옵션 추가:
  - 초급 (Beginner)
  - 중급 (Intermediate)
  - 고급 (Advanced)
  - 또는 "1급", "2급", "3급", "4급", "5급", "6급" (TOPIK 기준)
- 필수 항목: ✅ 체크

### 3단계: 폼 설정

1. **설정 탭 클릭**
   - 우측 상단 "설정" 아이콘 클릭

2. **응답 수집 설정**
   - "응답 수집" ✅ 체크
   - "이메일 주소 수집" (선택사항)
   - "제출 후 확인 메시지 표시" ✅ 체크
   - 확인 메시지: "감사합니다! 곧 연락드리겠습니다." / "Thank you! We'll contact you soon."

3. **프레젠테이션 설정**
   - "제출 후 링크 표시" ✅ 체크 (선택사항)
   - "진행 표시줄 표시" ✅ 체크

### 4단계: 폼 공유 링크 가져오기

1. **공유 버튼 클릭**
   - 우측 상단 "보내기" 버튼 클릭

2. **링크 복사**
   - "링크" 탭 선택
   - "URL 단축" 체크 해제 (전체 URL 필요)
   - 링크 복사

3. **임베드 코드 생성**
   - 복사한 링크에서 폼 ID 추출
   - 예: `https://docs.google.com/forms/d/[FORM_ID]/viewform`
   - 임베드 URL: `https://docs.google.com/forms/d/[FORM_ID]/viewform?embedded=true`

### 5단계: HTML에 적용

1. **폼 ID 확인**
   - 복사한 링크에서 `d/` 다음의 긴 문자열이 폼 ID입니다
   - 예: `1MMl-dAx7gt-liFXD0bWTOhoRAvsz38xxAWRn52ZcA4c`

2. **HTML 코드에 적용**
   ```html
   <iframe src="https://docs.google.com/forms/d/[여기에_폼_ID_입력]/viewform?embedded=true" 
           frameborder="0" 
           marginheight="0" 
           marginwidth="0" 
           style="position: absolute; top: 0; left: 0; width: 100%; height: 100%; border: none;">
       로드 중…
   </iframe>
   ```

### 6단계: 응답 확인

1. **응답 탭에서 확인**
   - 폼 편집 화면에서 "응답" 탭 클릭
   - 실시간으로 제출된 응답 확인 가능

2. **구글 시트로 연결 (선택사항)**
   - "응답" 탭에서 "스프레드시트에 연결" 클릭
   - 새 시트 생성 또는 기존 시트 선택
   - 모든 응답이 자동으로 시트에 저장됨

## 주의사항

- 폼 ID는 공개되어도 안전하지만, 폼 설정에서 "누구나 응답 가능"으로 설정되어 있는지 확인하세요
- 응답 데이터는 구글 계정의 구글 드라이브에 저장됩니다
- 개인정보 보호를 위해 GDPR 등 관련 법규를 준수하세요

## 문제 해결

- **폼이 표시되지 않을 때**: 폼 ID가 정확한지, `?embedded=true`가 URL 끝에 있는지 확인
- **응답이 저장되지 않을 때**: 폼 설정에서 "응답 수집"이 활성화되어 있는지 확인
- **모바일에서 폼이 잘리거나 작게 보일 때**: CSS의 `form-wrapper` 스타일 확인
